About

EventAdvisor


EventAdvisor is an app that streamlines the proccess of organizing and finding events. Its two key features are:

  • Letting users with accounts to create events with ease through facebook events integration
  • Allows all app users to browse events in a location and see published event details

For someone that wants to find something fun to do near them, EvendAdvisor is the perfect resource to use!
