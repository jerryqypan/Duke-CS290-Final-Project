About

EventAdvisor


EventAdvisor is an app that streamlines the process of organizing and finding events. Its two key features are:

  • Letting users with accounts to create events with ease through facebook events integration
  • Allowing all app users to browse events near a location and see published event details
  • Allowing all app users discover events nearby or in a certain area
  • Letting account holders create events with ease through facebook event integration

For someone who wants to find something fun to do, EventAdvisor is the perfect resource!
